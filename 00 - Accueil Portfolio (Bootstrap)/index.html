WOD generator

Une application JavaScript qui génére des entrainements

L'idée de départ est assez simple, basée sur le concept de Freeletics ou d'autres applications dans le même genre, mais adaptée au CrossFit. En effet, l'algorythme de ce genre d'application personnalise le training selon le profil; dans le CrossFit, c'est un entrainement unique tous les jours (WOD, pour Workout Of the Day) et on le réussit selon ses capacités. L'idée est donc de concevoir une application en JS qui générera un WOD tous les jours selon les exercices officiels du CrossFit et les WODs de référence, le tout selon le matériel disponible chez les pratiquants.

  • Date: A Venir

Jeu de plateforme

Un jeu de plateforme bien rétro comme il faut

Dans l'idée de diversifier mes projets, mais surtout de m'amuser avec le code et de pouvoir créer ce que j'ai en tête, je développerai seul mon propre jeu de plateforme comme on pouvait les connaitre dans les années 90. Ayant grandi avec ce type de jeu, c'est un peu un rêve que de pouvoir concevoir le mien, avec mon propre univers. Afin de rester dans un univers connu et de rendre mes projets utiles à mes compétences professionnelles, le jeu sera développé en javascript et prévu pour fonctionner sous navigateur web.
